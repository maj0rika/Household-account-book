# Roadmap v2: 레거시 안내 문서

이 문서는 2026-02-25 기준 v2 backlog와 실행 메모를 임시로 정리했던 레거시 문서다.

현재 저장소의 **단일 기준 계획 문서**는 [docs/implementation-plan.md](./implementation-plan.md)이며, 이 파일은 과거 링크 호환과 이관 안내를 위해서만 유지한다.

---

## 현재 기준 위치

- 현재 구현 Phase: [docs/implementation-plan.md](./implementation-plan.md)
- 확장 Phase: [docs/implementation-plan.md](./implementation-plan.md)
- 통합 backlog / 운영 메모 / 결정 로그: [docs/implementation-plan.md](./implementation-plan.md)
- 히스토리 인덱스: [docs/implementation-plan.md](./implementation-plan.md)

## 이관 범위

- 기존 `roadmap-v2`의 P0~P4 backlog는 `implementation-plan`의 `통합 backlog` 섹션으로 이관했다;
- 기존 Phase 1~6 실행 계획은 `implementation-plan`의 `backlog 실행 묶음` 섹션으로 이관했다;
- 기술 메모, 확정 의사결정, 구현 확인 메모도 `implementation-plan`로 옮겼다;

## 유지 이유

- 기존 `docs/history/`와 `todos/` 문서가 이 파일 경로를 참조하고 있어 파일 자체는 삭제하지 않는다;
- 새 계획, backlog, 운영 메모는 이 파일에 추가하지 않고 `implementation-plan`만 갱신한다;
