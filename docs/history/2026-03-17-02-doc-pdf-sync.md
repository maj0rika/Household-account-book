# 인터뷰 Markdown PDF 동기화

## 변경 내용

- `docs/`의 인터뷰용 Markdown 중 `docs/pdf/`에 없던 7개 문서를 PDF로 생성해 추가했습니다.
- 기존 PDF와 합쳐 인터뷰용 Markdown 10개 전부가 `docs/pdf/`에 대응되도록 맞췄습니다.

## 변경된 파일

- docs/pdf/fe-code-interview-supplement.pdf
- docs/pdf/interview-cheatsheet.pdf
- docs/pdf/interview-job-prep.pdf
- docs/pdf/interview-learning.pdf
- docs/pdf/interview-live-demo.pdf
- docs/pdf/interview-pressure-qa.pdf
- docs/pdf/interview-summary.pdf
- docs/implementation-plan.md

## 결정 사항

- `pandoc` 같은 별도 변환 도구를 추가하지 않고, 로컬 Playwright 렌더링으로 Markdown 기반 PDF를 생성했습니다.
- 기존 `docs/pdf/` 파일명 규칙을 그대로 따라 Markdown basename과 PDF basename을 일치시켰습니다.
