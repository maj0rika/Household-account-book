# 프롬프트 입력 플레이스홀더 정산 예시 추가

## 변경 내용

- 자연어 입력바의 순환 플레이스홀더에 `n/1`, `내가 냈고 내 몫`, `카톡 정산 스샷` 예시를 추가했다;
- 사용자가 정산 입력도 이 입력창에서 바로 시도할 수 있다는 점이 처음부터 보이도록 문구 범위를 넓혔다;

## 변경된 파일

- `src/components/transaction/NaturalInputBar.tsx` — 정산 관련 플레이스홀더 예시 추가

## 결정 사항

- 정산 예시는 총무 관점과 스크린샷 업로드 관점을 각각 하나씩 넣어, 텍스트 입력과 이미지 입력 둘 다 유도한다;
- 문구는 길이를 과도하게 늘리지 않고 한 화면에서 즉시 해석 가능한 자연어 예시만 추가한다;

## 다음 할 일

- 파싱 결과 시트에도 정산 입력 예시를 노출할지 검토한다;
