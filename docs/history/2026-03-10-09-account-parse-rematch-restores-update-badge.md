# 자산 부채 재매칭 시 업데이트 배지 복원

## 변경 내용

- 자산/부채 파싱 결과 시트에서 사용자가 매칭 키를 수정한 뒤 다시 정확히 일치하는 값으로 되돌리면 `업데이트` 배지가 다시 보이도록 수정한다.
- 매칭 키를 바꾸지 않은 편집에서만 사용자의 `신규` 선택을 유지하고, 키가 바뀐 뒤의 재매칭에서는 정확한 기존 계정 업데이트로 복귀시킨다.

## 변경된 파일

- docs/history/2026-03-10-09-account-parse-rematch-restores-update-badge.md
- docs/implementation-plan.md
- src/components/assets/AccountParseResultSheet.tsx

## 결정 사항

- 사용자의 수동 `신규` 선택은 금액 수정 같은 비매칭 편집에서는 유지하되, 이름/유형/세부 유형을 바꾼 뒤 다시 기존 계정과 정확히 맞아떨어지면 자동으로 `업데이트` 상태를 복구한다.

## 다음 할 일

- 수동으로 `신규`를 선택한 뒤 이름/유형/세부 유형을 바꿨다가 다시 맞추는 시나리오에서 배지가 `업데이트`로 복귀하는지 점검한다.
